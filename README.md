# CROB-Retreat

This repo includes a clickable entry page plus the outreach map and distressed ZIP coverage tools.

## View in browser
If you click an `.html` file inside the GitHub repo, GitHub shows the source code. To see the actual page, open the published site instead:

`https://sharonsmith-829.github.io/CROB-Retreat/`

## Files
- [index.html](index.html) — main entry page with links to the HTML views
- [outreach_dashboard.html](outreach_dashboard.html) — combined outreach map and distressed ZIP coverage page
- [outreach_heatmap.html](outreach_heatmap.html) — interactive map
- [zip_coverage_analysis.html](zip_coverage_analysis.html) — distressed ZIP analysis tool
- [requirements.txt](requirements.txt) — Python dependencies
- [geocode_heatmap.py](geocode_heatmap.py) — main script

## Quick start
Open [index.html](index.html) first, then click the page you want.

If you are already on GitHub, use the Pages URL above rather than clicking the file link.

## Notes
- The outreach map and ZIP coverage pages are static HTML files, so they open directly in a browser.
- If you want GitHub Pages later, the `index.html` file is already set up as the landing page.
